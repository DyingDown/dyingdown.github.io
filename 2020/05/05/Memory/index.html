Memory

2020-05-05
Essay 2 mins733 words

每个人都有梦想。

梦想是握在手中的彩色气球，等到实现的那一刻，它们会徜徉在头顶的天空，为你的人生画上五彩缤纷的美丽。于是，你为放飞梦想的一刻而拼命努力，不曾停滞。当然，为梦想而努力是一种幸福。这样的幸福会牢牢刻进心底，一辈子，都不会忘记。

梦想……我也曾经拥有过。

可当它有一天突然破碎掉的时候，我难过心痛却无法理直气壮的大声哭泣。我擦干泪水，静静的拾起梦想的碎片，作为失去它的惩罚，我把它和幸福一起装进了潘多拉的盒子里……永远不能再轻易开启。

……

音乐大楼。

就像沙漠中的绿洲突然冒出在辰纱面前，颇有艺术的白色建筑搭配透明的窗顶玻璃，虽然只有五层高，但十分气派。记忆中，从中学念了这所学校开始，她一次也没来过这里。

如同被召唤。

几乎能想象到音符跳跃在黑白相间的键盘上的情景。

回忆从远远的那一处乘风而来，变成了一把小小的钥匙停留在她的手心里，她拿着钥匙，站在种满风信子的门前，轻轻打开……

是那个童话中的小姑娘。

小姑娘穿着一身白色的纱裙，坐在三角架钢琴前熟练的弹奏。小姑娘的身边坐着她爸爸，爸爸一脸慈爱的看着小姑娘，一边打节奏，听着女儿稚嫩的弹唱。

你最喜欢的这旋律 在天空中响遍的口琴

天使拥抱着窗框里的画布 喂……这幅风景画 漂亮吗？

这是被风带来的……淡淡的花瓣……春之追忆

美丽的音色……咏唱的少女 鸟儿的鸣叫 指针的前进

与穹苍紧连的……流动的云……夏之追忆

美丽的音色……放歌的少女 ……蝉们的阵雨……

漂亮呢……你所说的景色……永远不忘……

美丽之物……为了将它采撷……生命才降诞到世上……

你所拥有的短暂的季节 尽管满是痛苦的雨的敲打……

“别担心哦”……笑着这么说道……你的样子 永远不忘……

小姑娘唱完后，得意的扬起下巴，充满期待和赞扬的看着爸爸，爸爸将小姑娘开心的抱起来旋转，爸爸说，纱纱，你是我最最骄傲的小公主。

画面渐渐融进了淡淡的雾色，她揉了揉眼。

她差点不记得，她也曾让爸爸感到骄傲过，可正是这个让他骄傲的女儿，最后却害死了他，亲手拆散了这个家，改变了所有人的命运。

曾经温柔的妈妈，曾经和她一样幸福的哥哥……
